Прогноз товарообігу роздрібної мережі (Retail Turnover Forecasting)

Проєкт містить інфраструктуру для аналізу історичних даних, розрахунку комерційних показників та побудови прогнозних моделей машинного навчання.

Структура проєкту та файлів

`dataset_turnover.csv` — основний індивідуальний датасет проєкту. Містить понад 500 рядків реальних історичних даних, адаптованих під специфіку товарообігу роздрібної мережі.
`Model market.ipynb` / `Model money.ipynb` / `Model pcs.ipynb`— інтерактивні блокноти Jupyter, що містять розрахунки, візуалізацію трендів та алгоритми навчання моделей.
Блокноти з суфіксом `_accuracy_calculation.ipynb` — скрипти для валідації результатів та оцінки точності отриманих прогнозів (RMSE, MAPE).
`three_phase_linear.py` — модуль із реалізацією базового математичного алгоритму трифазної лінійної моделі.

Специфікація вхідних даних (`dataset_turnover.csv`)

Таблиця оптимізована для аналізу динаміки продажів за категоріями товарів і містить такі ключові стовпці:
1.  `date` — звітний період (дата у форматі YYYY-MM-DD).
2.  `category_id` — унікальний ідентифікатор товарної категорії в роздрібній мережі.
3.  `revenue_retail` — фактичний обсяг товарообігу (виручка від реалізації) у грошовому виразі.
4.  `margin` — розрахований обсяг маржинального доходу (чистого прибутку від продажів) на основі середньої рентабельності категорії.
